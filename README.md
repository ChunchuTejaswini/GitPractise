# FB Clone Website

This is a simple Facebook homepage clone developed as part of my Git and GitHub learning journey. The project was created to practice version control concepts while working on a real-world frontend application.

The clone replicates the basic layout and user interface of Facebook, including the navigation bar, login section, news feed layout, sidebar, and responsive design elements using HTML and CSS.

## Features

- Facebook-inspired homepage design
- Responsive layout using HTML and CSS
- Clean and organized project structure
- Practice of Git version control operations such as:
  - Initializing a Git repository
  - Creating and switching branches
  - Making commits
  - Merging branches
  - Rebasing branches
  - Resolving merge conflicts
  - Working with tags
  - Pushing changes to GitHub

## Technologies Used

- HTML5
- CSS3
- Git
- GitHub

## Purpose

The primary goal of this project was not only to recreate the Facebook user interface but also to gain hands-on experience with Git and GitHub workflows. Throughout the development process, I practiced branching strategies, commit management, merging, rebasing, and repository management while maintaining the project in Git.

This project serves as a practical demonstration of both frontend development fundamentals and Git version control concepts.
